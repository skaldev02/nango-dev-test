## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm/yarn/pnpm
- Free Nango account

### Installation

```bash
# 1. Install dependencies
npm install

# 2. Configure environment
cp .env.local.example .env.local
# Edit .env.local with your Nango keys

# 3. Run development server
npm run dev

# 4. Open browser
# http://localhost:3000
```

**⚡ The page editor works immediately!**
