## 🐛 Common Issues & Solutions

### Issue: "Public key authentication is deprecated"
**Cause**: Still using old public key
**Solution**: 
1. Remove `NEXT_PUBLIC_NANGO_PUBLIC_KEY` from `.env.local`
2. Restart dev server
3. Clear browser cache

### Issue: "Failed to create connect session"
**Cause**: Invalid or missing secret key
**Solution**: 
1. Verify `NANGO_SECRET_KEY` in `.env.local`
2. Get new secret key from Nango dashboard
3. Restart server

### Issue: "Connection not found"
**Cause**: No connection exists for this user
**Solution**: 
1. Click "Connect" first to create connection
2. Then click "Fetch Data"

### Issue: OAuth popup blocked
**Cause**: Browser popup blocker
**Solution**: 
1. Allow popups for localhost
2. Try again

### Issue: "No OAuth credentials configured"
**Cause**: Integration not set up in Nango dashboard
**Solution**: 
1. Follow setup guide for the integration
2. Configure OAuth credentials in Nango dashboard
